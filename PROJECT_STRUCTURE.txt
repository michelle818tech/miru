📝 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════

MANAGER WORKFLOW (Weekly):
1. Log in
2. See Odoo metrics (auto-synced Sunday 5 AM)
3. Write weekly log:
   - "Which products did you focus on?" (AI checks for specificity)
   - "What actions did you take?" (AI checks for numbers/dates)
4. Submit → AI blocks if too vague, approves if specific
5. At month-end, submit WHAT % + HOW %

HR WORKFLOW:
1. Log in
2. See dashboard with all 4 managers
3. Review weekly logs + Odoo metrics
4. Write work ethics notes + warnings
5. Approve monthly scores
6. System calculates KPI tier (100%/70%/40%/0%)
