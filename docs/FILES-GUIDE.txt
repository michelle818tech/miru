MANAGER WORKFLOW

1. Manager logs in
2. Sees weekly Odoo metrics (auto-pulled):
   - Sales ₮
   - # transactions
   - Avg order value
   - Bundle rate
   - Refund rate
3. Writes 2 reflections:
   - "Which products focused on?" → MUST be specific
   - "What actions taken?" → MUST have numbers/dates
4. Submits → AI checks
   - ✓ APPROVED: specific products + numbers + dates
   - ✗ BLOCKED: "too vague", "need product names", etc.
5. At month-end, submits WHAT % + HOW %
